Tugas Pemrograman Dasar

Tugas Pemrograman Dasar ini melibatkan pembuatan program yang dapat menghitung percepatan, kecepatan, dan energi potensial. Dengan menggunakan program ini, pengguna dapat dengan mudah menghitung nilai-nilai tersebut dengan memberikan input yang sesuai.

No Soal

3 Buatkan program aplikasi perhitungan energi potensial menggunakan html.
